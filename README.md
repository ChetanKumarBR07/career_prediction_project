**Smart Career Counselling System which is an AI based web application to guide students to the best career options suitable for them based on their interests, skills and academic performance. The system implements a Decision Tree model to confront user inputs and yield personalized career recommendations. It includes an assessment with questionnaires and a resume analysis to improve the prediction accuracy. This integration with Flask allows for easy communication between the front and back end of the site while providing a pleasant user experience. Using data-driven decision-making, the project fills the gap between students and informed career choices. Some future work involves developing more sophisticated machine learning models and making the user experience more engaging. friendly interface.**
